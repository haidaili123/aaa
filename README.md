# 🌸 第七章可视化平台

> 3D图表、动画与统计地图的交互式学习网站

## 📖 项目简介

第七章可视化平台是基于《Python数据可视化之美》第七章内容开发的交互式学习网站，专注于3D图表制作、动画设计和统计地图绘制。网站采用豆沙粉主题设计，营造温馨优雅的学习氛围。

## 📊 功能模块

### 🌟 7.1 mplot3d绘制3D图表

基于matplotlib的mplot3d工具包，提供3D空间数据可视化功能。

#### 🎯 核心功能
- **3D散点图**: 三维空间星星分布可视化
- **3D线框图**: 网格化3D数据展示
- **3D曲面图**: 平滑曲面数据渲染

#### 🎮 交互控制
- **图表类型切换**: 散点图、线框图、曲面图实时切换
- **星点数量调节**: 支持10-100个星点动态调整
- **旋转视图**: 45度增量旋转，多角度观察3D数据
- **实时统计**: 显示星点总数、平均高度、旋转角度

#### 📝 学习要点
```python
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D
fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
ax.scatter(x, y, z, c='#C71585', marker='*', s=160)
```

### ✨ 7.2 animation制作动图

基于matplotlib.animation模块，创建动态可视化效果。

#### 🎯 核心功能
- **闪烁星星**: 透明度和缩放动画效果
- **正弦波动画**: 数学函数动态可视化
- **旋转动画**: 3D空间连续旋转效果

#### 🎮 交互控制
- **动画类型选择**: 闪烁、波动、旋转三种模式
- **速度调节**: 1-10级速度控制
- **播放控制**: 播放、暂停、重置功能
- **帧数统计**: 实时显示当前动画帧数

#### 📝 学习要点
```python
from matplotlib.animation import FuncAnimation
def animate(i):
    star.set_color('#C71585' if i % 2 else 'white')
    return star
ani = FuncAnimation(fig, animate, frames=None, interval=1000)
```

### 🗺️ 7.3 basemap绘制统计地图

基于mpl_toolkits.basemap，实现地理数据可视化。

#### 🎯 核心功能
- **气泡地图**: 城市人口分布气泡图
- **散点地图**: 地理位置散点分布
- **热力地图**: 数据密度热力可视化

#### 🎮 交互控制
- **地图类型切换**: 气泡图、散点图、热力图
- **数据点数量**: 50-500个城市数据动态调整
- **气泡大小**: 1-20级大小缩放控制
- **随机数据**: 一键生成随机地理数据

#### 📝 学习要点
```python
from mpl_toolkits.basemap import Basemap
m = Basemap(projection='stere', lat_0=90, lon_0=-105)
m.drawcoastlines()
m.drawstates()
m.scatter(x, y, s=population_size)
```
